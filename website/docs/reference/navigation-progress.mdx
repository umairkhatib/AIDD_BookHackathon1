# Course Navigation & Progress Tracking

## Overview

The Physical AI & Humanoid Robotics course includes a comprehensive navigation and progress tracking system to help students monitor their learning journey through all course modules. This system provides an intuitive interface for tracking completion status and navigating between different course components.

## Features

### Progress Tracking
- **Lesson Completion**: Mark individual lessons as complete
- **Module Progress**: Visual progress bars for each module
- **Overall Progress**: Complete course progress overview
- **Persistent Storage**: Progress saved locally in your browser

### Navigation System
- **Module-Based Organization**: Clear organization by course modules
- **Quick Access**: Direct links to all course content
- **Progress-Based Navigation**: See where you left off in your studies
- **Responsive Design**: Works on all device sizes

## How to Use

### Tracking Your Progress
1. Navigate to any lesson in the course
2. Complete the lesson content and exercises
3. Click the "Mark Complete" button in the course navigation panel
4. Your progress will be automatically saved

### Viewing Course Progress
- The main course navigation page shows overall progress
- Individual module progress bars display completion status
- Lesson lists show which content has been completed
- All progress is saved between sessions

### Navigation Features
- **Module Links**: Click any module title to go to the start
- **Lesson Links**: Direct access to any lesson in the course
- **Progress Indicators**: Visual feedback on completion status
- **Quick Jump**: Navigate to any part of the course instantly

## Progress Calculation

The system calculates progress based on:
- Number of lessons completed per module
- Overall course completion percentage
- Time spent on course materials
- Exercise completion status

## Data Privacy

All progress data is stored locally in your browser using localStorage:
- No personal data is collected
- Progress is not shared with others
- Data remains on your device only
- Clear your browser data to reset progress

## Troubleshooting

### Progress Not Saving
If your progress isn't saving:
1. Ensure cookies/localStorage are enabled in your browser
2. Try a different browser or clear your cache
3. Check if you're using a private/incognito window

### Navigation Issues
If navigation links don't work:
1. Refresh the page
2. Check your internet connection
3. Ensure JavaScript is enabled
4. Try a different browser
